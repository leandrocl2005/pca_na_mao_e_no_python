PCA na mo e no Python

PCA, ou principal component analysis uma tcnica para reduzir a dimenso de um conjunto de dados preservando, de certa forma, suas propriedades. Executar essa tcnica na mo ou sem ajuda de funes prontas um bom exerccio para aprender a fundo o que est por trs desse processo. Nesse tutorial faremos um exemplo de PCA na mo, sempre conferindo as contas com Python e depois faremos o mesmo exemplo usando a biblioteca Scikit-learn.

Pequeno exemplo de PCA para fazer na mo

Como utilizaremos o Python para conferir nossas contas, importaremos as bibliotecas necessrias.

# bibliotecas
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

Utilizaremos as seguintes matrizes para esse trabalho:

$$X = \left(\begin{array}[cc]\\ 1 & 2\\ 3 & 4\\ 1 & 3\\ 2 & 4\\ 2 & 3\\ 1 & 4 \end{array}\right)\ \ y = \left(\begin{array}[c] \\ 0\\ 1\\ 0\\ 1\\ 1\\ 1\end{array}\right)$$

A matriz $X$ representa as features, enquanto $y$ representa o target. Declararemos essas matrizes no Python:

In [2]:
# matrizes no Python
df = pd.DataFrame({
    'var1':[1,3,1,2,2,1],
    'var2':[2,4,3,4,3,4],
    'target': [0,1,0,1,1,1]
})
X = df.drop('target',1)
y = df['target']

Precisaremos das mdias de cada coluna de $X$. Para o clculo da mdia da coluna i basta fazer a seguinte conta:

$$\frac{\sum_j X_{ji}}{n}$$

com $n$ sendo o nmero de linhas da matriz $X$. Podemos conferir as contas no Python usando a biblioteca numpy:

In [3]:
# mdia por coluna de X
mean_vec = np.mean(X, axis=0)
print(mean_vec)

var1    1.666667
var2    3.333333
dtype: float64

Nossa prpria tarefa determinar a matriz de covarincia. Para isso, vamos usar uma matriz M auxiliar. Para determinar M subtraia a mdia da coluna i da coluna respectiva. Como no nosso exemplo todas as entradas de M esto multiplicadas por $\frac{1}{3}$, para ficar mais elegante, colocamos $\frac{1}{3}$ em evidncia.

$$M = \frac{1}{3}\left(\begin{array}[ll]\\ -2 & -4\\ \ \ 4 & \ \ 2\\ -2 & -1\\ \ \ 1 & \ \ 2\\ \ \ 1 & -1\\ -2 & \ \ 2 \end{array}\right)$$

Para realizar a mesma conta no Python, use o cdigo a seguir:

In [4]:
# subtraindo a mdia da respectiva coluna de X
M = X - mean_vec

A matriz de covarincia de $X$ dada por:

$$C = \frac{1}{n-1} M^T M$$

onde $n$ o nmero de linhas de $X$.

Logo, $$C = \frac{1}{3}\left(\begin{array}[cc] \\ 2 & 1 \\ 1 & 2\end{array}\right)$$

No Python:

In [5]:
# calculando matriz de covarincia
C = M.T.dot(M) / (X.shape[0]-1)

Prximo passo determinar auto-valores e auto-vetores da matriz de covarincia C. Os auto-valores $\lambda$ de C so raizes de $$p_C(x) = \det(C - xI)$$ onde I a matriz identidade.

Cada auto-valor $\lambda$ tem auto-vetores associados. Apesar de auto-vetores nunca serem nulos, se inclurmos o vetor nulo no conjunto dos auto-vetores associados e chamarmos esse novo conjunto de $V_{\lambda}$, temos que $V_{\lambda}$ o conjunto soluo do seguinte sistema: $$(C - \lambda I)X = 0$$ Tomaremos geradores unitrios de $V_{\lambda_1}$ e de $V_{\lambda_2}$, para $\lambda_1=1$ e $\lambda_2 = \frac{1}{3}$, a saber, tomaremos os seguintes auto-vetores: $$\left\{ \left(\frac{\sqrt{2}}{2}, -\frac{\sqrt{2}}{2}\right), \left(\frac{\sqrt{2}}{2},\frac{\sqrt{2}}{2}\right)\right\}.$$ Essa escolha no nica e isso pode dar pequenas diferenas nos resultados finais.

No Python:

# determinando auto-valores e auto-vetores
autovalores, autovetores = np.linalg.eig(C)

# imprimindo auto-valores de C
print("Auto-valores:")
print(autovalores)
print()

Agora, ordenaremos os auto-valores do maior para o menor. Assim $$\lambda_1 = 1\text{ e }\lambda_2 = \frac{1}{3}.$$ No Python:

In [8]:
# ordenando em ordem descrecente
pares_de_autos = [
    (
        np.abs(autovalores[i]),
        autovetores[:,i]
    ) for i in range(len(autovalores))
]
pares_de_autos.sort()
pares_de_autos.reverse()

Nosso prximo passo calcular o quanto cada componente (auto-vetores) esto representando (de certa forma) a variabilidade dos nossos dados. Para isso usaremos duas medidas.

A varincia explicada de cada auto-valor $\lambda$ dada por: $$\frac{\lambda}{\sum_j \lambda_j}$$

A varincia explicada acumulada de cada auto-valor $\lambda_i$ dada por: $$\frac{\sum_{j\leq i}\lambda_j}{\sum_j \lambda_j}$$

No Python:

In [9]:
# calculando a varincia explicada e a varincia explicada cumulativa
total = sum(autovalores)
var_exp = [
    (i / total)*100 for i in sorted(
        autovalores, reverse=True
    )
]
cum_var_exp = np.cumsum(var_exp)

Vejamos de forma sumarizada que temos at o momento:

Bom, isso tudo! Considerando os dados originais como pares ordenados em $\mathbb{R}^2$, pois $X$ possui apenas duas colunas, ao mudar da base cannica para a base composta pelos auto-vetores encontrados, teremos uma nova representao do conjunto $X$. Vejamos graficamente:

# considerando base de auto-vetores
n_componentes = 2 # projetamos nas 2 (duas) primeiras componentes
autovetores = [p[1] for p in pares_de_autos]
A = autovetores[0:n_componentes] 
X = np.dot(X,np.array(A).T)
new_df = pd.DataFrame(X,columns=['pc1','pc2'])

    new_df, vars = ['pc1','pc2'], hue='target', diag_kind="hist"

  • Os dados originais foram rotacionados por um ngulo de 45 uma vez que os auto-vetores so perpendiculares e obtidos dos eixos x e y por uma rotao de 45 no sentido anti-horrio.
  • Nossos dados possuiam 2 colunas, mas poderiam possuir bem mais. Alm disso escolhermos 2 componentes para visualizao grfica, esse nmero tambm pode ser mais, desde que seja menor que o nmero de colunas.
  • A ideia do PCA diminuir a dimenso do conjunto de dados original escolhendo um nmero menor de componentes para projeo sobre estas.
  • O processo de reduo de dimenso facilita processamento e armazenamento de dados.

PCA para reduo de dimenso em imagens

Outra aplicao a reduo de dimenso em imagens. Para esse exemplo, escolhemos a imagem Pssaro mineiro barulhento retirada do site Pixabay. A imagem foi transformada em escala de cinza utilizando o software GIMP.

O nmero 0.99 passado como parmetro para o mtodos PCA diz que queremos um nmero de componentes que nos garanta 99% de varincia explicada cumulativa. Observe que para isso, utilizou-se 145 componentes. Isso uma reduo imensa, uma vez que a original possuia 1899 colunas. Queremos ver, caso necessrio recuperar a imagem, como essas imagens ficam com a dimenso reduzida. Para isso sero necessrios duas funes auxiliares.

In [24]:
def pca_with_var_exp(X, var_exp=0.99):
    pca = PCA(var_exp) # varincia explicada de 0.99
    lower_dimension_data = pca.fit_transform(X)
    print(lower_dimension_data.shape)
    approximation = pca.inverse_transform(lower_dimension_data)
    return approximation

def plot_subplot(X, i):
    plt.subplot(3,2,i)
    plt.imshow(X, cmap="gray")
    plt.xticks([])
    plt.yticks([])

A primeira funo reduz a imagem X mantendo var_exp de varincia explicada cumulativa. Compararemos 3 valores: 0.99, 0.95 e 0.90.
